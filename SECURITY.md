# Security Policy

## Supported Versions

`ocrs-cjk` is currently in early development. Security fixes are provided for the latest released version and the `main` branch.

| Version | Supported |
| ------- | --------- |
| latest  | ✅ |
| older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in `ocrs-cjk`, please do **not** open a public GitHub issue.

Instead, please use [GitHub's private vulnerability reporting](../../security/advisories/new) to report it confidentially, or contact the maintainer privately.

Please include as much information as possible:

- A description of the vulnerability
- Steps to reproduce
- Affected version or commit
- Potential impact
- Suggested fix, if available

## Response Timeline

The maintainer will try to acknowledge reports within 7 days.

Security issues will be prioritized based on severity, exploitability, and impact.

## Responsible Disclosure

Please allow reasonable time for investigation and remediation before publicly disclosing the issue.

We appreciate responsible security research and coordinated disclosure.
